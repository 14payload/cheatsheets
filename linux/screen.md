### create new session
```
screen -S <name>
```

### log session to file
```
screen -S <name> -L -Logfile <file>
```

### detach
```
CTRL + a, d
```

### kill session
```
CTRL + a, k
```

### scroll
```
CTRL + a + Esc -> use arrow keys
```

### go back from scrolling
```
q or Esc
```

### list session
```
screen -list
```

### reattach
```
screen -r <name>
```

### detach session
```
screen -d <name>
```

### switch windows
```
CTRL + a + n
```

### new window
```
CTRL + a + c 
```
